mxnet安装

发表于 2016-09-06

linux注意事项

发表于 2016-05-17

tar -xf自动识别格式没必要写全

yum install 先update。apt-get同理。

find 查找 which查找命令对于文件（windows下叫where）

sudo nohup nice -n -20 cmd & nohup不允许terminal发送hangup给命令进程（防止关闭terminal导致进程结束）nice)指定命令进程优先级-20最高 19最低 命令后加&可以后台运行命令，使得我们可以继续在terminal输入，当然这样就看不到命令输出了，如果要看到输出可以重定向输出到文件然后使用tail来跟踪(tail -n 100 -f file)
